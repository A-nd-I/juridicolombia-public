# Contact

Use this repository for product transparency, roadmap discussion, screenshots, and public feedback.

## Product Feedback

Open a GitHub Issue for:

- Feature requests.
- UX feedback.
- Documentation improvements.
- Official-source link corrections.
- Bug reports that do not include sensitive information.

## Security Reports

Do not open public issues for security vulnerabilities. Follow `SECURITY.md`.
