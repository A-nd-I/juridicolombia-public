# Landing Materials

This folder contains public-facing product material for JuridiColombia. All general communication is written in English. Colombian legal-domain names remain in Spanish when they refer to specific legal sources, such as `Código Civil`, `Código de Comercio`, `Código General del Proceso`, and `artículo`.

## Contents

- `index.html`: visual landing page for demos and GitHub Pages.
- `PRODUCT-DESCRIPTION.md`: full product narrative.
- `SALES-NOTES.md`: pitch notes and objection handling.
- `INDEX.md`: quick navigation by audience and use case.
- `OVERVIEW.md`: summary of this landing package.

## Recommended Use

- Use `index.html` as the public landing page.
- Use `PRODUCT-DESCRIPTION.md` for partners, advisors, and product reviews.
- Use `SALES-NOTES.md` before demos or discovery calls.
- Use `INDEX.md` to decide what material to read first.

## Public Positioning

JuridiColombia is a privacy-first Chrome Extension for fast legal reference in Colombia. It helps legal professionals search embedded public legal texts directly from the browser, with offline availability and minimal permissions.

## Language Rule

- Product, engineering, roadmap, issues, and contact copy: English.
- Specific Colombian legal-domain names: Spanish.
